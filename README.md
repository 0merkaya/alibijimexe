# alibijimexe

Ali bijim exe tamamen zararsız bir şaka yazılımıdır.
Oluşacak hiçbir yasal sorumluluğu kabul etmiyorum.
Yazılım sadece 1 dk boyunca çalışıp ekranın renklerini değişitirip resimler çıkartır. Bu süre boyunca eğer UAC izni verip administrator olarak çalıştırırsanız klavye ve mouse bu 1 dakikalık süre boyunca çalışmayacaktır.
